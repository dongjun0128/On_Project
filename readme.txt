자바 프로젝트 import 방법
1. Eclipse JAVA EE버전으로 실행
2. json-simple-1.1.1.jar파일 다운
3. OnProject.war파일 import
4. 프로젝트 우클릭 후 Properties 클릭
5. JAVA Build Path - Libraries 클릭
6. ModulePath 클릭 후 Add External JARs..에서 json-simple-1.1.1.jar 파일 추가
7. Apply and Close
